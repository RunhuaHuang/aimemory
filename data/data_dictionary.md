# Data Dictionary: AI Memory Boundary Fit CB

All files contain participant-level study data. Composite variables are computed as the mean of their item columns. Each study file also includes `Age`, `Gender`, `Grade`, and `Testing_Channel` fields.

- `study1_memory_data.csv`: Memory_Type, Memory_Code, MBF, Trust_Item1-3, Purchase_Item1-3, Continue_Item1-3.
- `study2_memory_data.csv`: Memory_Type, Memory_Code, MBF, Relational_Warmth, Surveillance_Concern, Trust_Item1-3.
- `study3_memory_data.csv`: Sensitivity, Memory_Type, Trust_Item1-3.
- `study4_memory_data.csv`: Memory_Type, Control_Interface, MBF, Trust_Item1-3.
- `study5_memory_data.csv`: Memory_Type, MBF, Trans_Item1-3, DataShare_Item1-3, Rel_Item1-3.
- `study6_memory_data.csv`: Memory_Type, Memory_Code, MBF, PrivacyTrust_Item1-3, Loyalty_Item1-3, OptIn_Item1-3, PlatformTrust_Item1-3.
